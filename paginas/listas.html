LISTAS EN HTML

Las listas HTML permiten a los desarrolladores web agrupar un conjunto de elementos relacionados en listas.

Lista HTML desordenada

Una lista desordenada comienza con la <ul>etiqueta. Cada elemento de la lista comienza con la <li>etiqueta. Los elementos de la lista se marcarán con viñetas (pequeños círculos negros) de forma predeterminada: Esta sería la forma en que se hace y el resultado de como quedaría.
